# 🌐 LinguaApp

> Proyecto de desarrollo aplicacion web enfocado en el aprendizace de nuevos idiomas.

---

## 📌 Descripción

Este proyecto consiste en una aplicación web desarrollada con NodeJS, Electron y React, cuyo objetivo es Brindar una plataforma para aprender nuevos Idiomas..

---

## 🛠️ Tecnologías utilizadas

### Frontend

* HTML5
* CSS
* JavaScript
* React

### Backend

* Node.js

### Base de datos

* MySQL

* 📊 Interfaz intuitiva
* ⚡ Diseño responsive
